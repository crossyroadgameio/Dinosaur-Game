The Dinosaur Game (often called Chrome Dino or Dino Game) was released on September 6, 2014 and developed by Sebastien Gabriel, Alan Bettes, and Edward Jung. It originally appeared inside Google Chrome whenever the internet went offline, quickly becoming a beloved casual classic enjoyed by millions.

You control a running T-Rex moving through a desert filled with obstacles. The goal is simple: survive as long as possible while the game speeds up over time.
